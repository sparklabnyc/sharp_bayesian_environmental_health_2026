  3. Fit and interpret advanced environmental models in INLA

Introduction

We will use the Dengue incidence during 2009-2024, in Brazil, at the micro-region level (645 areas), as downloaded from DATASUS.

For that analysis, we included 5,201,154 confirmed dengue cases in Sao Paolo. We retrieved mean, min and max monthly temperature, total monthly precipitation and mean monthly relative humidity from ERA5.Population was retrieved from the 2011 census.

We will use Bayesian hierarchical models to quantify the effect of meteorology while adjusting for spatial, temporal and spatiotemporal autocorrelation.

WRITE THE MODEL!!

To include random effects in INLA, you need to create IDs.

# seasonality
data_Brazil$id_seasonal <- data_Brazil$month
# long-term trends
data_Brazil$id_year <- data_Brazil$year - min(data_Brazil$year) + 1
# spatial effects
data_Brazil$id_space <- data_Brazil$code_muni %>% as.factor() %>% as.numeric()

We need to make sure that the spatial shp is alligned with the data structure. That means that the region with id_space 1 needs to be the 1st row of the shapefile, the regions with id_space 2, needs to be the 2nd row of the shapefile, etc.

Select the priors:

# Priors
hyper.iid <- list(theta = list(prior="pc.prec", param=c(1,0.01)))
hyper.bym <- list(theta1 = list('PCprior', c(1,0.01)), theta2 = list('PCprior', c(0.5, 0.5)))

Define the formula

mod1 = cases ~ 1 + 
  offset(log(population)) +  
  
  # long-term trends
  f(id_year, model = "rw1", hyper = hyper.iid, constr = TRUE) + 
  # seasonality
  f(id_seasonal, model = "rw2", hyper = hyper.iid, constr = TRUE) + 

  # spatial effect
  f(id_space, model = "bym2", graph='W.adj', hyper = hyper.bym, constr = TRUE) + 
  
  # meteorological linear terms
  rh + t2m_max + tp_sum

and the INLA call:

Spatially varying temperature

In INLA you cannot use twice the same id. If needed you need to create a new column with a different name:

data_Brazil$id_space2 <- data_Brazil$id_space

Define the formula

mod2 = cases ~ 1 + 
  offset(log(population)) +  
  
  # long-term trends
  f(id_year, model = "rw1", hyper = hyper.iid, constr = TRUE) + 
  # seasonality
  f(id_seasonal, model = "rw2", hyper = hyper.iid, constr = TRUE) + 

  # spatial effect
  f(id_space, model = "bym2", graph='W.adj', hyper = hyper.bym, constr = TRUE) + 
  
  # meteorological linear terms
  rh + t2m_max + tp_sum +
  
  f(id_space2, t2m_max, model = "bym2", graph='W.adj', hyper = hyper.bym, constr = TRUE)

If you have defined a linear effect of the temperature, you would need to add constraints for the SVC. The interpretation would be about spatial deviations of the overall linear term. If you do not include a linear effect, then the SVC is the total effect, including the average across the area.

Closing remarks

In this lab session, we have explored how to fit advanced Bayesian models using INLA. We looked at linear spatially varying coefficients and distributed non-linear lag models.

We analyzed real-world data on confirmed dengue cases in Brazil from 2009 to 2024 to quantify the impact of meteorological factors. Our approach allowed the effects of temperature and rainfall to vary across space, and to exhibit delayed and non-linear patterns. We accounted for underlying spatiotemporal trends and interactions, and interpreted the results using a range of visualizations to illustrate the estimated effects.
